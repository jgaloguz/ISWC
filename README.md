# ISWC
This repository holds the C++ codes used in the galactic cosmic ray modulation project of the US portion of the annual International Space Weather Camp (ISWC), hosted at the University of Alabama in Huntsville.

The project materials are separated into (deterministic) single particle motion and (stochastic) cosmic ray modulation simulations, the codes for which are respectively contained in the `single_particle_motion` and `gcr_modulation` folders.
The folder `docs` contains a PDF tutorial of the course content and useful figures/animations which will be discussed during the lectures.
C++ codes are mostly complete with a few snippets missing where `\\ TODO` comments are found.
Students are responsible for reading the manual, found in `docs/GCR_Tutorial.pdf`, and completing these codes to replicate figures found in each section.
